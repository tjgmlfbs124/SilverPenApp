# 모바일 코딩게임 프로젝트
## 사업개요
- 각종 컨텐츠와, 코딩을 할수있는 기능을 구현한 어플리케이션
- 회원가입 시스템
- 활동을 하면 활동로그를 서버에 전송
- 블루투스를 통해 로봇을 제어할수 있는 컨트롤러 ui 제공

## 개발개요
- Application Name : "실버펜"
- Develop Tool : Android Studio
- compileSdk Version : 29
- minSDK Version : 21

- 마지막 개발 : 2019-12-18
- 디자인 : 사원 김혜인
